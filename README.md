#software-lab

## Description
This project demonstrates embedding PHP code inside HTML,
and running it with GitHub Actions to automate build and testing.

## Features
- PHP + HTML project
- GitHub repository with proper commits
- GitHub Actions CI for automated testing

## Technologies Used
- PHP
- HTML
- GitHub Actions

## CI/CD
GitHub Actions automatically runs PHP syntax check
and executes the PHP file on every push or pull request.
